# User Journeys — B2B Floral Marketplace

## Retail Florist Buyer Journey

### Discovery Phase
1. **Entry points**: Direct search, supplier recommendation, industry directory
2. **First impression**: Homepage showcases trending stems, seasonal availability, trust badges
3. **Account creation**: Business verification (tax ID, resale certificate), credit application

### Browsing & Selection
1. Navigate to category (Roses > Garden Roses > David Austin)
2. Apply filters: stem length 50cm+, Grade A, available within 3 days
3. Compare 3 suppliers on price, MOQ, shipping cost
4. View product detail: farm origin, harvest date, vase life estimate, customer reviews
5. Add to cart with quantity (25 bunches × 10 stems)

### Checkout & Payment
1. Review cart with line-item pricing
2. Select delivery date and time window
3. Choose shipping method (refrigerated truck, air freight)
4. Apply account credit or enter payment (Net-30 terms)
5. Confirm order, receive confirmation email with tracking

### Post-Purchase
1. Track shipment in real-time
2. Receive delivery, inspect quality
3. Report issues via photo upload (damaged stems, wrong grade)
4. Rate supplier, leave review
5. Save order as template for reorder

## Wholesale Sales Rep Journey

### Daily Workflow
1. **Morning**: Review overnight orders, check inventory alerts
2. **Mid-day**: Process RFQs, generate custom quotes for large accounts
3. **Afternoon**: Follow up on pending quotes, upsell slow-moving inventory
4. **End of day**: Sync orders with ERP, prepare next-day shipments

### Quote Generation Flow
1. Receive RFQ notification (email + in-app)
2. Open customer profile: purchase history, credit status, preferred pricing tier
3. Build quote: select products, apply customer discount, add freight estimate
4. Preview quote PDF, add personal note
5. Send quote via platform messaging or email
6. Track quote status: viewed, accepted, expired, counter-offered

### Relationship Management
1. View customer health score (order frequency, payment history, support tickets)
2. Schedule follow-up calls in integrated calendar
3. Log call notes and next actions
4. Set alerts for customer milestones (anniversary, volume threshold)

## Farm/Grower Supplier Journey

### Inventory Listing
1. Log in to supplier dashboard
2. Batch upload harvest forecast CSV (variety, quantity, grade, cut date)
3. Set pricing: base price, volume tiers, promotional discounts
4. Add product photos (stem, bunch, box packaging)
5. Publish listings, set availability window

### Order Fulfillment
1. Receive order notification
2. Print packing slip with QR code
3. Pick and pack stems per order specs
4. Apply cold chain labels, schedule carrier pickup
5. Mark order as shipped, upload tracking number

### Payment & Analytics
1. View pending receivables by customer
2. Track payment status (pending, overdue, paid)
3. Review sales analytics: top products, peak order days, buyer geography
4. Export reports for accounting

## Journey Pain Points & Design Opportunities

| Journey Stage | Pain Point | Design Opportunity |
|---------------|------------|-------------------|
| Browsing | Too many clicks to compare suppliers | Side-by-side comparison modal |
| Checkout | Unclear freight costs until final step | Real-time freight calculator in cart |
| Reorder | Finding previous order takes too long | "Reorder" button on dashboard homepage |
| Quote | Manual PDF generation is slow | One-click quote builder with templates |
| Inventory | Bulk upload errors unclear | Inline validation with error highlighting |
| Payment | Tracking overdue invoices is fragmented | Unified receivables dashboard |
